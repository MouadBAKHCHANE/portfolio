## üíª Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MouadBAKHCHANE/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

- `/public`: Static assets (images, fonts).
